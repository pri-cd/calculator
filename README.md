# 🧮 Modern Calculator

A clean, responsive, and highly efficient calculator application built to handle everything from daily quick arithmetic to fluid mathematical expressions. Designed with a modular structure and an intuitive user interface.

---

## 🚀 Features

* **Core Arithmetic:** Seamless handling of addition, subtraction, multiplication, and division.
* **Responsive UI/UX:** Modern, pixel-perfect design that adapts flawlessly across mobile, tablet, and desktop viewports.
* **Keyboard Support:** Input operators and digits directly from your physical keyboard for rapid utility.

---

## 🛠️ Tech Stack

* **Frontend Core:** HTML5, CSS3, JavaScript (ES6+)
* **Framework Options:** Vanilla
* **Styling:** Custom CSS Flexbox.
